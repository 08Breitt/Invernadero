Portafolio del Proyecto

En este proyecto, desarrollamos un sistema de monitoreo y control para invernaderos, que permite optimizar el crecimiento de las plantas mediante el control de variables ambientales clave. El sistema incluye:

Descripción del Problema

La empresa agrícola GreenGrow Solutions enfrentaba problemas relacionados con el control eficiente del ambiente en sus invernaderos, lo que afectaba el crecimiento de las plantas y el uso de recursos como el agua y la energía.

Metodología

Utilizamos la instrumentación virtual, mediante la integración de sensores, microcontroladores Arduino y comunicación con LabVIEW, para obtener datos en tiempo real y tomar decisiones automáticas de control.

Resultados Obtenidos

El sistema diseñado logró una optimización significativa del ambiente, garantizando condiciones ideales para el crecimiento de las plantas y mejorando la eficiencia en el uso de recursos.
